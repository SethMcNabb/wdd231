Recipe Central Site Plan

Site Name

Recipe Central. This website is just that! A place to find new recipies, save good-looking ones for later, and try new things!

Site Purpose

Recipe Central's purpose is to answer the age old question "What should we have for dinner?" by allowing users to see new recipies and save intesting ones for later, we can help you find your new favorite meals!

Scenarios

  • “What is for dinner?”
  • “I want to try this recipe later, can I save it for later?”

Color Scheme

Primary Color: Olive Green (#556B2F) for headers and navigation.

Secondary Color: Warm Beige (#F5F5DC) for background and section accents.

Typography

Roboto Condensed will be used for all fonts, in varying sizes depending on the element being used
